Architectural rules

Every line of code in this pipeline obeys these.

RULE 01

No raw source reaches the LLM

The LLM consumes typed AST nodes, symbol slices, def-use chains, and business-rule predicates — never raw COBOL. This is what prevents hallucinations.

RULE 02

Every node has a stable UUID

SHA-256 of file + kind + line + col, first 32 hex, formatted as a UUID. Same source byte-for-byte produces identical UUIDs forever — rebuilds are reproducible.

RULE 03

Cross-refs use UUIDs, never names

A paragraph calling another paragraph stores from_uuid → to_uuid, not "A" → "B". That's what makes the bundle a true graph instead of a text search.
